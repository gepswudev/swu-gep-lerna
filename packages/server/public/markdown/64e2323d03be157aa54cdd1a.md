# ผลงานทางวิชาการ
**1.  งานวิจัย**


**1.1  บทความวิจัยตีพิมพ์ในวารสารวิชาการระดับชาติและนานาชาติ (อยู่ในฐานข้อมูลของ สกอ.)**

ภารดี กำภู ณ อยุธยา.(2565) การศึกษาและพัฒนารูปแบบ การจัดการเรียนรู้เพื่อเสริมสร้างการรับรู้ความสามารถของตนเองด้านความคิดสร้างสรรค์ในนิสิตวิชาชีพครู. วารสารมนุษย์ศาสตร์และสังคมศาสตร์มหาวิทยาลัยนครพนม, 12(1), 212-227.

พาสนา จุลรัตน์, ธรรมโชติ เอี่ยมทัศนะ, และภารดี กำภู ณ อยุธยา.(2562). การสร้างแบบวัดการคิดเชิงบริหารของนักเรียนวัยรุ่นระดับมัธยมศึกษาตอนปลาย. วารสารสุทธิปริทัศน์, 33(107), 97-110.

พาสนา จุลรัตน์, ธรรมโชติ เอี่ยมทัศนะ, และภารดี กำภู ณ อยุธยา.(2562). การสร้างแบบวัดการคิดเชิงบริหารของนักเรียนวัยรุ่นระดับมัธยมศึกษาตอนต้น. วารสารวิชาการและวิจัยสังคมศาสตร์, 14(3). 47-62.

พัชราภรณ์ ศรีสวัสดิ์, ครรชิต แสนอุบล, ธรรมโชติ เอี่ยมทัศนะ, ภารดี กำภู ณ อยุธยา และ บดินทร์ นาวงศรี (2565). การเสริมสร้างสมรรถนะการให้คำปรึกษาสำหรับนักเรียนของครูแนะแนวโดยนวัตกรรมการจัดการเรียนรู้ : การฝึกอบรม. วารสารวิจัยทางการศึกษา คณะศึกษาศาสตร์ มหาวิทยาลัยศรีนครินทรวิโรฒ, 17(2). กรกฎาคม-ธันวาคม, 82-92.

**2.  ตำรา/หนังสือ/บทความทางวิชาการ**

ภารดี กำภู ณ อยุธยา, และธรรมโชติ เอี่ยมทัศนะ. (2563). การเรียนรู้ที่มีประสิทธิภาพและบทบาทของครูในสังคมแห่งความรู้. วารสารวิชาการและวิจัยสังคมศาสตร์, 15(1), 1-16.

**3. ข้อมูลประสบการณ์ / ความถนัด / ความสนใจพิเศษ**

3.1 อาจารย์สังกัดภาควิชาการแนะแนวและจิตวิทยาการศึกษา คณะศึกษาศาสตร์

3.2 วิทยากร ผู้เขียนบทความ ที่ปรึกษา ฯลฯ สาขาวิชา จิตวิทยาการศึกษา

1) คณะกรรมการพัฒนาหลักสูตรโครงการบ่มเพาะต้นกล้าค่านิยม 12 ประการ ของผู้เรียนระดับการศึกษาขั้นพื้นฐานในพื้นที่หมู่บ้านอาสาพัฒนาป้องกันตนเอง (อพป.)

2) วิทยากรอบรมหัวข้อ “การจูงใจกับการจัดการชั้นเรียน” ให้แก่คณะครูโรงเรียนอำนวยวิทย์ จังหวัดสมุทรปราการ

3) วิทยากรอบรมกิจกรรมโครงการค่ายครู คณะศึกษาศาสตร์ ระหว่างวันที่ 15-17 สิงหาคม 2562 ณ จังหวัดนครนายก

4) ทีมวิทยากรผู้ออกแบบกิจกรรมและการฝึกอบรม “โครงการกิจกรรมเสริมเพื่อพัฒนาทักษะชีวิตของนักศึกษาครูรัก(ษ์) ถิ่น รุ่นที่ 1 (2564) ของกองทุนเพื่อความเสมอภาคทางการศึกษา (กสศ.)
